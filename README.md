OrzMC私服官方网站

## 使用方式

```bash
# 安装最新版Hugo
brew install hugo
# 拉取仓库代码到本地
git clone https://github.com/OrzGeeker/OrzMCSite.git
# 切换到仓库目录下
cd OrzMCSite
# 同步网站主题子仓库模块
git submodule sync
# 本地预览网站
./preview
```
