# 服务端部署

- [基岩版服务端下载](https://www.minecraft.net/en-us/download/server/bedrock)

```bash
$ unzip bedrock-server-1.16.210.06.zip -d [target_dir]
$ LD_LIBRARY_PATH=. 
$ ./bedrock_server
```
ubuntu下查看端口打开情况：`$ lsof -i:19132`

服务器安全规则打开UDP协议对应端口

服务端端口UDP: 
- IPv4: `19132`
- IPv6: `19133`

# 基岩版客户端国际版下载

- 美区`Apple ID`申请
- 淘宝购买minecraft兑换码
- 登录美区`Apple ID`，使用兑换码安装国际版Minecraft
