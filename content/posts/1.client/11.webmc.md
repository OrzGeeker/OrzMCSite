## WebMC 浏览器客户端

- [Minecraft Classic](https://classic.minecraft.net/): Mojang 官方提供的基础功能 Minecraft 网页体验版，仅支持方块放置，适合快速体验。

- [Eaglercraft](https://eaglercraft.com/): WebMC 客户端的典型方案，可几乎完整运行 Java 版客户端功能，但因违反 Mojang EULA 协议被禁止，因为它反编译并重新打包了原 Java 版客户端代码，使其可以在浏览器里运行。

- [mcjs](https://www.mc.js.cool/): 中文版 WebMC，完成度较高，版本较老，仅支持 `1.8.8/1.12`

---

Mojang 官方提供的 Web 版本功能较少，建议吸收 Eaglercraft 的实践成果，探索合规的 Web 版客户端方案。
